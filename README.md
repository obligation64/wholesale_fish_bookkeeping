# 水产批发记账系统

专为水产批发业务设计的专业记账管理系统，支持客户管理、产品管理、销售记录、支出管理、数据统计和报表分析。

**🌤️ 已集成腾讯云开发(CloudBase)云存储，支持数据云端备份和同步！**

**💾 支持三种存储模式：本地SQLite、云开发、混合模式，可灵活切换！**

## 🎯 系统特点

### 批发业务特色
- **客户管理**: 区分客户类型（长期客户、批发商、零售客户、餐饮客户）
- **产品分类**: 按水产种类分类（鱼类、虾类、蟹类、贝类、软体类等）
- **斤数统计**: 精确记录每笔交易的斤数和单价
- **付款状态**: 跟踪客户付款状态（已付款、未付款、部分付款）
- **欠款管理**: 自动统计每个客户的累计欠款

### 支出分类
- 薪资 - 员工工资管理
- 车辆费用 - 运输车辆相关费用
- 油费 - 燃油费用
- 店面 - 店铺租金、维护等
- 水电 - 水电费用
- 进货成本 - 采购水产品的成本
- 运输费 - 外包运输费用
- 包装费 - 包装材料费用
- 其他 - 其他杂项支出

## 💡 使用指南

### 添加销售记录
1. 切换到"销售管理"标签页
2. 选择客户（或点击"+ 添加客户"快速添加）
3. 选择产品（或点击"+ 添加产品"快速添加）
4. 输入斤数和单价（系统自动计算总金额）
5. 选择付款状态
6. 选择日期
7. 点击"保存销售记录"

### 添加支出记录
1. 切换到"支出管理"标签页
2. 选择支出分类（薪资、车辆费用、油费等）
3. 输入金额
4. 选择支付方式
5. 添加备注（可选）
6. 点击"保存支出记录"

### 管理客户
1. 切换到"客户管理"标签页
2. 填写客户信息
3. 选择客户类型
4. 点击"保存客户"
5. 在表格中查看每个客户的累计销售额和累计欠款

### 管理产品
1. 切换到"产品管理"标签页
2. 填写产品信息
3. 选择产品分类
4. 设置建议单价（可选）
5. 填写供应商信息（可选）
6. 点击"保存产品"

### 生成报表
1. 切换到"报表分析"标签页
2. 选择日期范围
3. 点击"生成报表"
4. 查看汇总数据和图表
5. 点击"导出报表"下载CSV文件

## 💾 数据存储

### 三种存储模式

系统支持灵活的存储模式切换：

| 模式 | 说明 | 特点 |
|------|------|------|
| **local** | 本地模式 | 仅使用SQLite，快速、简单、无网络依赖 |
| **cloud** | 云模式 | 仅使用云开发，支持多设备协作 |
| **hybrid** | 混合模式 | 本地为主，自动云端同步 |

**详细说明**: 请查看 [存储模式说明.md](./存储模式说明.md)

## 🌤️ API接口

### 存储模式管理
- `GET /api/storage/mode` - 获取当前存储模式
- `POST /api/storage/mode` - 切换存储模式

### 云开发同步 (cloud/hybrid模式可用)
- `POST /api/cloud/sync/all` - 全量同步到云端
- `POST /api/cloud/sync/incremental` - 增量同步到云端
- `GET /api/cloud/sync/status` - 查看同步状态
- `POST /api/cloud/restore/:tableName` - 从云端恢复数据
