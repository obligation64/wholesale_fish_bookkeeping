# 水产批发记账系统

专为水产批发业务设计的专业记账管理系统，支持客户管理、产品管理、销售记录、支出管理、数据统计和报表分析。

## 🎯 系统特点

### 批发业务特色
- **客户管理**: 区分客户类型（长期客户、批发商、零售客户、餐饮客户）
- **产品分类**: 按水产种类分类（鱼类、虾类、蟹类、贝类、软体类等）
- **斤数统计**: 精确记录每笔交易的斤数和单价
- **付款状态**: 跟踪客户付款状态（已付款、未付款、部分付款）
- **欠款管理**: 自动统计每个客户的累计欠款

### 支出分类
- 薪资 - 员工工资管理
- 车辆费用 - 运输车辆相关费用
- 油费 - 燃油费用
- 店面 - 店铺租金、维护等
- 水电 - 水电费用
- 进货成本 - 采购水产品的成本
- 运输费 - 外包运输费用
- 包装费 - 包装材料费用
- 其他 - 其他杂项支出

## 🚀 快速开始

### 方法1: 直接打开
```bash
cd /home/zhoumeihua/wholesale_fish_bookkeeping
# 在浏览器中打开 index.html
```

### 方法2: 使用本地服务器（推荐）
```bash
# 使用 Python
python -m http.server 8000

# 或使用 Node.js
npx http-server -p 8000
```

然后在浏览器访问: `http://localhost:8000`

## 💡 使用指南

### 添加销售记录
1. 切换到"销售管理"标签页
2. 选择客户（或点击"+ 添加客户"快速添加）
3. 选择产品（或点击"+ 添加产品"快速添加）
4. 输入斤数和单价（系统自动计算总金额）
5. 选择付款状态
6. 选择日期
7. 点击"保存销售记录"

### 添加支出记录
1. 切换到"支出管理"标签页
2. 选择支出分类（薪资、车辆费用、油费等）
3. 输入金额
4. 选择支付方式
5. 添加备注（可选）
6. 点击"保存支出记录"

### 管理客户
1. 切换到"客户管理"标签页
2. 填写客户信息
3. 选择客户类型
4. 点击"保存客户"
5. 在表格中查看每个客户的累计销售额和累计欠款

### 管理产品
1. 切换到"产品管理"标签页
2. 填写产品信息
3. 选择产品分类
4. 设置建议单价（可选）
5. 填写供应商信息（可选）
6. 点击"保存产品"

### 生成报表
1. 切换到"报表分析"标签页
2. 选择日期范围
3. 点击"生成报表"
4. 查看汇总数据和图表
5. 点击"导出报表"下载CSV文件

## 💾 数据存储

所有数据存储在浏览器的 localStorage 中，包括：
- 销售记录 (fish_sales)
- 支出记录 (fish_expenses)
- 客户信息 (fish_customers)
- 产品信息 (fish_products)

**注意**: 清除浏览器数据会丢失所有记录，建议定期导出报表备份。

## 📝 使用场景

### 日常批发业务
- 每天记录销售到不同客户的各类水产品
- 记录每笔销售的斤数、单价和总金额
- 跟踪客户的付款状态和欠款情况

### 成本管理
- 记录员工薪资支出

- 统计进货成本和运输费用

- 了解主要客户销售情况
- 计算净利润和经营效率

- 区分不同类型的客户
- 跟踪客户的购买历史
- 管理客户欠款情况
- 分析客户价值

## 🎯 最佳实践

1. **定期导出数据**: 每周或每月导出报表备份
2. **准确记录斤数**: 确保斤数记录准确，便于统计
3. **及时更新付款状态**: 客户付款后及时更新状态
4. **合理设置产品单价**: 设置建议单价便于快速录入
5. **分类管理**: 正确选择客户类型和产品分类
