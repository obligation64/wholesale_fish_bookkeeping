# 快速启动指南

## 第一次使用

### 1. 安装依赖

```bash
npm install
```

### 2. 验证配置

```bash
node verify-config.js
```

这会检查:
- Node.js版本
- 云开发配置
- 所需文件是否存在
- 依赖是否已安装

### 3. 测试云开发连接(可选)

如果需要使用云同步功能,先运行测试:

```bash
node test-cloud.js
```

### 4. 启动服务

```bash
npm start
```

服务将在 http://localhost:3000 启动

## 一键启动脚本

使用自动安装和启动脚本:

```bash
chmod +x install-and-start.sh
./install-and-start.sh
```

脚本会自动:
1. 检查Node.js环境
2. 安装依赖
3. 提示配置云开发(可选)
4. 启动服务

## 使用云开发功能

### 配置密钥

**方法1: 使用环境变量**
```bash
export TCB_SECRET_ID="你的SecretId"
export TCB_SECRET_KEY="你的SecretKey"
npm start
```

**方法2: 使用.env文件**
```bash
cp .env.example .env
# 编辑.env文件,填入密钥
```

### 手动同步数据

启动服务后,可以使用以下接口同步数据:

**全量同步:**
```bash
curl -X POST http://localhost:3000/api/cloud/sync/all
```

**增量同步:**
```bash
curl -X POST http://localhost:3000/api/cloud/sync/incremental
```

**查看同步状态:**
```bash
curl http://localhost:3000/api/cloud/sync/status
```

## 故障排查

### 问题1: 依赖安装失败

**解决方法:**
```bash
# 清除npm缓存
npm cache clean --force

# 删除node_modules
rm -rf node_modules package-lock.json

# 重新安装
npm install
```

### 问题2: 云开发连接失败

**原因**: 缺少API密钥或网络问题

**解决方法**:
1. 检查网络连接
2. 配置腾讯云API密钥(参考上面的配置方法)
3. 确认环境ID是否正确

### 问题3: 端口已被占用

**解决方法**:
```bash
# 使用其他端口启动
PORT=3001 npm start
```
