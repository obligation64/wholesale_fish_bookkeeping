# 快速参考 - 存储模式切换

## 三种模式速查

| 模式 | 命令 | 适用场景 | 需要密钥 |
|------|-------|----------|-----------|
| **local** | `export STORAGE_MODE=local` | 本地使用、开发测试 | ❌ 否 |
| **hybrid** | `export STORAGE_MODE=hybrid` | 日常+备份 | ✅ 是 |
| **cloud** | `export STORAGE_MODE=cloud` | 多人协作 | ✅ 是 |

## 快速启动

```bash
# 默认(local模式)
npm start

# 指定模式
export STORAGE_MODE=hybrid
npm start
```

## API接口

```bash
# 查看当前模式
curl http://localhost:3000/api/storage/mode

# 切换到混合模式
curl -X POST http://localhost:3000/api/storage/mode \
  -H "Content-Type: application/json" \
  -d '{"mode":"hybrid"}'
```

## 云同步 (hybrid/cloud模式)

```bash
# 全量同步
curl -X POST http://localhost:3000/api/cloud/sync/all

# 增量同步
curl -X POST http://localhost:3000/api/cloud/sync/incremental

# 查看状态
curl http://localhost:3000/api/cloud/sync/status
```

## 环境变量

```bash
# .env文件
STORAGE_MODE=local              # 存储: local/cloud/hybrid
TCB_SECRET_ID=your-id          # 云密钥ID
TCB_SECRET_KEY=your-key        # 云密钥Key
```

## 推荐配置

### 个人使用
```bash
export STORAGE_MODE=hybrid
# 本地快速 + 云端备份 + 离线可用
```

### 团队使用
```bash
export STORAGE_MODE=cloud
# 云端共享 + 集中管理
```

### 开发测试
```bash
export STORAGE_MODE=local
# 无需配置 + 快速开发
```
