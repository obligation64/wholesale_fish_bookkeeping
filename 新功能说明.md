## 🌟 核心功能

### 1. 云数据库集成
- 自动连接腾讯云开发
- 创建和管理数据集合
- 支持所有CRUD操作
- 批量数据处理

### 2. 数据同步
- **全量同步**: 将所有本地数据同步到云端
- **增量同步**: 只同步新增或更新的数据
- **自动同步**: 默认每30秒自动执行增量同步

### 3. 数据备份与恢复
- 支持手动触发备份
- 支持从云端恢复数据
- 同步状态跟踪

### 4. 集合管理
系统在云端创建以下集合:
- customers (客户)
- products (产品)
- sales (销售记录)
- expenses (支出记录)
- monthly_expenses (月度固定支出)
- inventory (库存)
- change_logs (变更日志)
- sync_status (同步状态)

## 🚀 使用方法

### 安装依赖
```bash

```

### 验证配置
```bash
node verify-config.js
```

### 测试云连接

### 启动服务

```bash

curl -X POST http://localhost:3000/api/cloud/sync/all

# 增量同步

# 查看状态
```


## 💡 使用场景

### 场景1: 数据备份
定期执行全量同步,将本地数据备份到云端,防止数据丢失。

### 场景2: 多设备同步
在多个设备上使用同一环境ID,通过增量同步实现数据实时同步。

### 场景3: 数据恢复
当本地数据丢失或损坏时,从云端恢复历史数据。

### 场景4: 远程访问
通过云开发控制台,可以在任何地方查看和管理数据。

## ⚙️ 同步配置

可在 `config.js` 中修改同步参数:

```javascript
sync: {
    enabled: true,      // 是否启用自动同步
    interval: 30000,    // 同步间隔(毫秒)
    batchSize: 100      // 批量上传数量
}
```
